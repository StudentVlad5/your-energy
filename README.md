# Your Energy - Fitness Application

**Your Energy** — це веб-застосунок для любителів спорту, який допомагає
знаходити різноманітні вправи для тренувань, фільтрувати їх за групами м'язів
або обладнанням, переглядати детальну інформацію та зберігати улюблені вправи.

Проект реалізовано на ванільному **JavaScript** з використанням зборщика
**Vite**.

## **Демонстрація:** https://studentvlad5.github.io/your-energy/

## Основний функціонал

- **Головна сторінка (Home):**

  - Герой-блок з мотиваційним заголовком та описом здорового способу життя.
  - Список **категорій вправ** (Exercises) з подальшим переглядом деталей.
  - Фільтрація вправ за:
    - м’язами (Muscles),
    - частинами тіла (Body parts),
    - обладнанням (Equipment).
  - **Quote of the day / Daily norm of sports** — інформаційний блок із
    рекомендаціями щодо щоденної активності.

- **Сторінка Favorites:**

  - Перегляд вправ, доданих користувачем у вибране.
  - Можливість повернутися до вправи та працювати з нею повторно.

- **Модальні вікна:**

  - **Modal exercises** — детальна інформація по вибраній вправі (опис,
    параметри, інструкції).
  - **Rating modal** — форма для оцінювання (рейтингу) вправи.
  - **Confirm / Cancel modal** — підтвердження дій користувача (наприклад,
    скасування введених даних).

- **Форма підписки:**

  - Поле для e-mail.
  - Валідація введених даних (на стороні клієнта).
  - Повідомлення користувачу про успішну/неуспішну відправку.

- **UI-елементи:**
  - Навігація між вкладками **Home** та **Favorites**.
  - Соцмережі (Facebook, Instagram, YouTube) у футері.
  - Адаптивна верстка для зручної роботи на різних розмірах екрану.

## Технологічний стек

Проект створено з використанням сучасних веб-технологій та бібліотек:

- **Core:** HTML5, CSS3 (PostCSS), JavaScript (ES6+).
- **Build Tool:** [Vite](https://vitejs.dev/) - для швидкої розробки та
  оптимізованої збірки.
- **HTTP Client:** [Axios](https://axios-http.com/) - для взаємодії з backend
  API.
- **Notifications:** [IziToast](https://izitoast.marcelodolza.com/) - для
  красивих спливаючих повідомлень.
- **Styling:** Normalize.css, кастомні стилі, адаптивна верстка (Mobile / Tablet
  / Desktop).
- **Icons:** SVG Spirtes / Inline SVG.

### Залежності (Dependencies)

Проект використовує наступні пакети:

- `axios`: ^1.12.2
- `izitoast`: ^1.4.0
- `normalize.css`: ^8.0.1
- `vite-plugin-html-inject`: ^1.1.2 (для розбиття HTML на паршали)

## Функціонал

- Головна сторінка (Home)

- Герой (Hero): Відображення мотиваційного блоку.

- Цитата дня: Отримання надихаючої цитати з API (кешується в LocalStorage на
  добу).

- Фільтри вправ

- Картки вправ: Відображення списку вправ з пагінацією.

- Модальне вікно вправи:

- Додавання/видалення з "Favorites" (localStorage).

- Можливість поставити оцінку (Rating) з валідацією email.

- Сторінка обраного (Favorites) Перегляд вправ, збережених користувачем.

- Адаптивна верстка карток.

- Видалення вправ зі списку.

Всі дані зберігаються в браузері користувача (LocalStorage), тому список не
зникає після перезавантаження.

Футер (Footer) Форма підписки на розсилку (валідація email:
^\w+(\.\w+)?@[a-zA-Z_]+?\.[a-zA-Z]{2,3}$).

Посилання на соціальні мережі.

Додаткові фічі (UI/UX) Loader: Спіннер під час завантаження асинхронних запитів.

Scroll Up: Кнопка для швидкого повернення нагору сторінки.

Mobile Menu: Бургер-меню для мобільних пристроїв.

## Запуск локально

1. Клонувати репозиторій:

   ```bash
   git clone https://github.com/StudentVlad5/your-energy.git
   cd your-energy

   ```

2. Встановити залежності: npm install

3. Запустити дев-сервер: npm run dev

4. Зібрати продакшн-білд: npm run build

5. Переглянути зібраний білд локально: npm run preview
